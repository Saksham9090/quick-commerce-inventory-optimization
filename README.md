# quick-commerce-inventory-optimization
Dark Store Inventory Optimization Dashboard using SQL and PowerBI

## Project Overview
This project analyzes grocery order data to understand demand patterns and optimize inventory management for quick commerce dark stores.

## Tools Used
- SQL
- Power BI
- DBeaver

## Key Insights
- Peak order demand between 10 AM – 3 PM
- Produce department drives the highest number of orders
- Bananas and fresh produce are the fastest moving SKUs

## Dashboard
The Power BI dashboard visualizes demand trends, top selling products, and department-level insights.
